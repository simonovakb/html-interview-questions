# HTML semantics

Ниже приведён список вопросов и кратких ответов по теме «HTML Семантика» - зачем нужна семантика в вёрстке, какие теги считаются семантическими, влияние на SEO, доступность, структуру интерфейса и лучшие практики использования.

---

## Вопросы:

<details>
  <summary>1. Что такое семантика в HTML и зачем она нужна?</summary>

  Семантика — это смысловое описание структуры контента с помощью соответствующих тегов. 

  Семантические теги упрощают интерпретацию страницы поисковиками, скринридерами и адаптируют интерфейс для разных устройств.

</details>

<details>
  <summary>2. Приведите примеры семантических и несемантических тегов.</summary>

  Семантические: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`, `<figure>`, `<figcaption>`, `<mark>`, `<time>`, `<address>`.


  Несемантические: `<div>`, `<span>`, `<b>`, `<i>`.
</details>

<details>
  <summary>3. Чем семантические теги отличаются от стилизующих?</summary>

  Семантические теги несут смысл, описывают структуру и части страницы, а стилизующие (`<b>`, `<i>`, `<font>`, `<center>`) отвечают только за внешний вид.

</details>
