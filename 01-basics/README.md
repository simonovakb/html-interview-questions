  <summary><h3>Что такое <em>HTML</em> и для чего он используется?</h3></summary>

  **HTML (HyperText Markup Language)** — язык разметки, который задаёт структуру и семантику веб‑страницы.

  - Что делает:
    - Формирует структуру контента: заголовки, параграфы, списки, секции (`<header>`, `<footer>`, `<section>`).
    - Описывает формы и взаимодействие пользователя (`<form>`, `<input>`, `<label>`).
    - Встраивает медиа и ссылки (`<img>`, `<video>`, `<a>`).
    - Обеспечивает семантику и доступность (помогает скрин‑ридерам и поисковым системам).


  - Взаимодействие с другими технологиями:
    - В паре с **CSS** отвечает за внешний вид.
    - В паре с **JavaScript** добавляет поведение и динамику.
    - Современный стандарт — **HTML5** (лучше поддержка мультимедиа и семантики).

  > Короткий ответ: HTML — это язык разметки для описания структуры веб‑страниц и их | семантики; в связке с CSS и JavaScript он формирует интерфейс и поведение веб‑приложений.
